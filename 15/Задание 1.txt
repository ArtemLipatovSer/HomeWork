1) Создайте класс "SortedList", который имеет свойство массив и методы для добавления и
  удаления элементов из массива, а также для сортировки массива в порядке возрастания. 
  Класс должен также иметь метод для поиска элемента в массиве.


2)Создайте класс "SocialMediaPlatform", который имеет следующие методы:

	Метод для создания нового аккаунта пользователя, который принимает в качестве аргументов имя пользователя, адрес электронной почты и пароль.
	Метод для входа в систему, который принимает в качестве аргументов адрес электронной почты и пароль

3)Создайте класс "OnlineStore", который имеет следующие методы:
 	 
	Метод для добавления товара в корзину, который принимает в качестве аргументов идентификатор товара, его название, цену и количество.
  	Метод для удаления товара из корзины по его идентификатору.
  	Метод для подсчета общей суммы товаров в корзине.
